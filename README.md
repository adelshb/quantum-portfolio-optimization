# Portfolio Optimization via Variational Quantum Eigensolver Optimization

Qiskit implementation of a Portfolio Optimization via Variational Quantum Eigensolver (VQE) optimization.

 ## Requirements
 * Python 3.8+
 * Qiskit
 * Gekko
 * CVXPY
 * MOSEK
